# DevOps Daily Tips Telegram Bot

Telegram bot delivering daily DevOps tips, incident postmortem summaries, tool-of-the-week spotlights, and on-call readiness prompts. Commands include /tip for today's DevOps best practice, /tool for a random infrastructure tool deep-dive, /postmortem to run an interactive 5-whys exercise, /oncall for pre-shift readiness checklist, /kubectl for random Kubernetes command of the day, /terraform for IaC pattern spotlight, and /alert for configuring daily digest at chosen time. Covers SRE, platform engineering, CI/CD, observability, and cloud cost optimization. Self-hostable with Docker, python-telegram-bot, MIT license.

## Setup
1. Set `TELEGRAM_BOT_TOKEN` in `.env`
2. `pip install -r requirements.txt`
3. `python main.py`
